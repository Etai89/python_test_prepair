הדפסה לפלט

הדפסה נעשית באמצעות print() - זוהי הפקודה היחידה להדפסה במבחן!

נקודות חשובות:
  • תמיד עם סוגריים ()
  • טקסט בתוך גרשיים או גרש-גרש
  • משתנים ללא גרשיים
  • ניתן להדפיס כמה דברים ברצף עם פסיק
print("hello") # הדפסת טקסט x = 5 print(x) # הדפסת משתנה print("הערך הוא:", x) # הדפסה מעורבת print(x, "זוגי") # כמה ערכים ביחד

קלט מהמשתמש

עיקרון קריטי: input() תמיד מחזיר מחרוזת! אם צריך מספר - חובה להמיר עם int() או float()
מחרוזת:
x = input()
לטקסט, שמות, מילים
מספר שלם:
x = int(input())
למספרים ללא נקודה עשרונית
מספר עשרוני:
x = float(input())
למספרים עם נקודה עשרונית
דוגמה נפוצה בבחינות:
# קלט שם (מחרוזת) name = input() # קלט גיל (מספר שלם) age = int(input()) # קלט ציון (מספר עשרוני) grade = float(input())

תנאים ואופרטורים

נקודתיים חובה! אחרי כל if חייבים להיות נקודתיים :
if תנאי: קוד שיתבצע אם התנאי נכון עוד קוד (עם אינדנטציה!)

אופרטורי השוואה

== שווה בדיוק
לא = בודד!
!= לא שווה
ההפך מ-==
> גדול מ
ללא שוויון
< קטן מ
ללא שוויון
>= גדול או שווה
כולל שוויון
<= קטן או שווה
כולל שוויון

אופרטורים לוגיים - הבנה מעמיקה

  • and - שני התנאים חייבים להיות נכונים יחד. אם אחד לא נכון, התוצאה False
  • or - מספיק שאחד מהתנאים נכון. רק אם שניהם לא נכונים התוצאה False
דוגמאות מהמבחן:
# בדיקת טווח - שני תנאים יחד if x > 5 and x < 10: print("בטווח 5-10") # תנאי רחב - אחד מהתנאים מספיק if x > 7 or x < 12: print("מחוץ לטווח 7-12") # בדיקת שוויון/אי-שוויון if x != 50: print("לא 50") if name == "דני": print("שלום דני!")

אופרטורים מתמטיים

הבדל קריטי: / ו-% עושים דברים שונים לחלוטין!
/ חילוק רגיל
מחזיר תוצאה עשרונית
30/3 = 10.0
10/3 = 3.333...
% שארית חלוקה
מחזיר מה שנשאר אחרי החלוקה
12 % 7 = 5
30 % 3 = 0
איך לזכור את ההבדל:
# חילוק רגיל - כמה פעמים נכנס + שברים 10 / 3 = 3.333... # 3 שלמים ועוד קצת # שארית - מה נשאר אחרי חלוקה שלמה 10 % 3 = 1 # 3 נכנס 3 פעמים (9), נשאר 1 # דוגמאות נוספות: 15 % 4 = 3 # 4*3=12, נשאר 3 20 % 5 = 0 # 5*4=20, לא נשאר כלום

בדיקת זוגי/אי-זוגי - המושג החשוב ביותר

עיקרון: מספר זוגי מתחלק ב-2 ללא שארית. אי-זוגי משאיר שארית 1.
# זוגי: שארית 0 כשמחלקים ב-2 if x % 2 == 0: print("זוגי") # אי-זוגי: שארית 1 כשמחלקים ב-2 if x % 2 == 1: print("אי-זוגי") # או עם else (יותר נפוץ בבחינות): if x % 2 == 0: print("זוגי") else: print("אי-זוגי")

מחרוזות - נושא קריטי במבחן

חשוב לזכור: מחרוזת = רצף תווים (כמו מערך). האינדקס הראשון תמיד 0, לא 1!
דוגמה לאינדקסים:
s = "HELLO" # אינדקס: 0 1 2 3 4 # H E L L O print(s[0]) # H - התו הראשון print(s[4]) # O - התו האחרון print(s[-1]) # O - התו האחרון (דרך אחרת)
len(s)
כמה תווים במחרוזת
len("hello") = 5
s.upper()
הופך לאותיות גדולות
"abc" → "ABC"
s.lower()
הופך לאותיות קטנות
"XYZ" → "xyz"
s.count('x')
כמה פעמים תו מופיע
"hello".count('l') = 2
דוגמאות נפוצות במבחנים:
# קבלת שם והדפסת האות הראשונה name = input() print(name[0].upper()) # האות הראשונה באות גדולה # ספירת כמה פעמים אות מופיעה text = "hello world" print(text.count('o')) # תדפיס 2 # בדיקת אורך מחרוזת password = input() if len(password) >= 8: print("סיסמה חזקה")

טבלת אמת - המפתח להבנת אופרטורים לוגיים

איך לזכור:
  • and - כמו כפל: אפס אחד הורס הכל (False אחד = False)
  • or - כמו חיבור: מספיק אחד חיובי (True אחד = True)
תנאי A תנאי B A and B A or B
True True True True
True False False True
False True False True
False False False False
דוגמה מעשית:
# נניח x = 8 # תנאי A: x > 5 → True (8 > 5) # תנאי B: x < 10 → True (8 < 10) # x > 5 and x < 10 → True and True → True # x > 5 or x > 15 → True or False → True # x < 3 and x > 0 → False and True → False

שגיאות נפוצות - חשוב לא לפספס!

השגיאות שגורמות לכישלון במבחן:
  • נקודתיים חסרים: לשכוח : אחרי if - השגיאה הכי שכיחה!
  • המרה חסרה: לכתוב x = input() במקום x = int(input()) כשצריך מספר
  • בלבול אופרטורים: להתבלבל בין / (חילוק) ל־% (שארית)
  • בלבול השוואה: לכתוב = במקום == בתנאים
  • אינדקס שגוי: לחשוב שהאינדקס הראשון הוא 1 במקום 0
  • גרשיים שגויים: להתבלבל מתי להשתמש בגרשיים ומתי לא
דוגמאות לשגיאות ותיקונים:
# ❌ שגוי if x > 5 # חסרים נקודתיים x = input() # יהיה מחרוזת, לא מספר if x = 5: # השוואה שגויה print x # חסרות סוגריים # ✅ נכון if x > 5: # עם נקודתיים x = int(input()) # עם המרה למספר if x == 5: # השוואה נכונה print(x) # עם סוגריים

דוגמאות קוד

בדיקת זוגי/אי-זוגי

x = int(input()) if x % 2 == 0: print("even") else: print("odd")

תנאי מורכב

x = int(input()) if x > 5 and x < 10: print("inside") else: print("outside")

פעולות על מחרוזות

s = input() print(s[0].upper()) print(len(s)) print(s.count('o'))

טיפים להכנה יעילה למבחן

אסטרטגיית הכנה

שלב 1 - חזרה על היסודות (יום לפני המבחן):
  • קיראו את כל הדף הזה מתחילה לסוף
  • תרגלו כל דוגמה בראש לפני שאתם רואים את התשובה
  • שיננו את טבלת האמת - היא קריטית!
  • תכתבו מזכרת קטנה עם השגיאות הנפוצות
שלב 2 - תרגול מעשי (3-4 ימים לפני):
  • כתבו קוד בעצמכם! אל תסתפקו בקריאה - פתחו מחשב וכתבו
  • תרגלו בדיקות זוגי/אי-זוגי עד שזה יהיה אוטומטי
  • כתבו 5 תוכניות קטנות עם input ו-print
  • תרגלו פעולות על מחרוזות - len(), upper(), count()
  • בדקו את עצמכם בטבלת האמת - תמציאו תרחישים ותבדקו
שלב 3 - סימולציה (יום המבחן):
  • קיראו כל שאלה פעמיים לפני שאתם עונים
  • זהו מילות מפתח: "זוגי" = % 2, "אורך" = len()
  • בדקו תמיד: נקודתיים אחרי if, סוגריים ב-print
  • אל תמהרו! רוב השגיאות הן מחוסר קשב, לא מחוסר ידע

תרגילים אחרונים לפני המבחן

תכתבו את התוכניות האלה בלי לראות תשובות:
  1. תוכנית שקולטת מספר ואומרת אם הוא זוגי או אי-זוגי
  2. תוכנית שקולטת שם ומדפיסה את האות הראשונה באות גדולה
  3. תוכנית שבודקת אם מספר נמצא בטווח 10-20
  4. תוכנית שסופרת כמה פעמים האות 'a' מופיעה במילה
  5. תוכנית שבודקת אם מספר גדול מ-100 או קטן מ-10
רק אחרי שתכתבו - בדקו עם הדוגמאות בדף!

תרגילים נוספים לתרגול

תרגילים למתחילים

תרגיל 1 - הדפסה וקלט בסיסי:
# כתבו תוכנית שקולטת שם ואומרת שלום name = input() print("שלום", name)
תרגיל 2 - בדיקת גיל:
# קלטו גיל ובדקו אם המשתמש בגר age = int(input()) if age >= 18: print("בגר") else: print("קטין")
תרגיל 3 - בדיקת ציון:
# קלטו ציון ובדקו אם עבר (מעל 60) grade = int(input()) if grade >= 60: print("עבר") else: print("נכשל")

תרגילים מתקדמים

תרגיל 4 - בדיקת טווח מספרים:
# בדקו אם מספר נמצא בין 50 ל-100 num = int(input()) if num >= 50 and num <= 100: print("בטווח") else: print("מחוץ לטווח")
תרגיל 5 - בדיקת חלוקה:
# בדקו אם מספר מתחלק ב-3 או ב-5 num = int(input()) if num % 3 == 0 or num % 5 == 0: print("מתחלק") else: print("לא מתחלק")
תרגיל 6 - עבודה עם מחרוזות:
# קלטו מילה והדפיסו אותה כולה באותיות גדולות word = input() print(word.upper()) print("אורך המילה:", len(word))

תרגילים לבחינה

תרגיל 7 - משולב מתקדם:
# קלטו שם ובדקו אם הוא מתחיל באות גדולה name = input() if name[0].upper() == name[0]: print("השם מתחיל באות גדולה") else: print("השם מתחיל באות קטנה")
תרגיל 8 - ספירת תווים:
# קלטו משפט וספרו כמה רווחים יש בו sentence = input() spaces = sentence.count(' ') print("כמות רווחים:", spaces)
תרגיל 9 - תרגיל מורכב:
# קלטו מספר ובדקו אם הוא חיובי וזוגי num = int(input()) if num > 0 and num % 2 == 0: print("המספר חיובי וזוגי") else: print("המספר לא חיובי וזוגי יחד")
תרגיל 10 - אתגר:
# קלטו שם ובדקו אם האות האחרונה זהה לראשונה name = input() if name[0].lower() == name[-1].lower(): print("האות הראשונה והאחרונה זהות") else: print("האותיות שונות")

משימות לתרגול עצמאי

נסו לפתור בעצמכם לפני שאתם מסתכלים על הפתרונות למעלה:
  1. קליטת שני מספרים: כתבו תוכנית שקולטת שני מספרים ומדפיסה את הגדול מביניהם
  2. בדיקת סיסמה: כתבו תוכנית שקולטת סיסמה ובודקת אם היא ארוכה מ-8 תווים
  3. חישוב ממוצע: קלטו שלושה ציונים וחשבו את הממוצע שלהם
  4. בדיקת תו: קלטו מחרוזת ותו, ובדקו אם התו מופיע במחרוזת
  5. משחק ניחושים: המחשב "חושב" על מספר (למשל 42), המשתמש מנחש ואתם אומרים אם נכון
